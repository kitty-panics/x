# X

判断并解压文件。

## 使用

1. 安装 **filetype** 包
    + `pip install filetype`
2. 将 `x` 放进 **PATH** 中。

解压文件只需执行下面的命令即可。

```Shell
x 文件名
```

## 支持格式

+ [X] zip
+ [X] rar
+ [X] 7z
+ [X] tar
+ [X] gz
+ [X] tgz
+ [X] bz
+ [X] bz2
+ [X] xz
+ [X] z
+ [ ] lz
+ [ ] deb
+ [ ] exe
